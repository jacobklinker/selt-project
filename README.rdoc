Deploying to local environment:
1.) bundle install
2.) rake db:create
3.) rake db:migrate
4.) rake db:seed 
5.) rails server -p $PORT -b $IP


Creating a Heroku instance:
1.) heroku create
2.) git push heroku master
3.) heroku run rake db:migrate
4.) heroku run rake db:seed


Deploying to Heroku:
1.) commit any changes
2.) git push heroku master
